This folder provides replication material for the paper. 

Code is fully reproducible and follows the IPO protocol.

Input -> Data
Processing -> Rmd scripts
	1. Data -> cleans the ESS
	2. Measures -> creates measures and print descriptives
	3. Models -> Run the models and print figures of the Article
Output -> Collection of all figures and tabels we show in the Article and in the Supplement. 


To reproduce the paper: 

1. Install R
2. Clone this repository (=download it locally)
3. Click on the RProject icon ("Lon_turn.R")
4. Run files sequentially (Data -> Measures -> Models) or in isolation
